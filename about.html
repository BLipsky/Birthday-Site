In the Honors Advanced Web Design class, the Monthly Website #3 project tasked students with creating an interactive and visually appealing website focused on historical events related to their birthdays across different years. The project outlined specific objectives, starting with the requirement to research and compile information on five historical events occurring on the student's birthday, supported by proper citations in a Google Doc. The design phase involved using Figma to create layouts for both Desktop and Mobile screens, emphasizing a visually appealing interface with Bootstrap components for responsiveness. The website's functionality was to be developed using HTML, CSS, and JavaScript in Visual Studio Code, with meaningful commits on GitHub to showcase the creative process. The design included five buttons that changed the site's HTML elements to display different historical events, each accompanied by a title, informational text, and at least one image. The project also emphasized content integration, requiring proper citation of images and quotations sourced from the researched materials. Additionally, the project incorporated a footer with a link to the student's GitHub repository, providing a pathway for the audience to explore the codebase. The GitHub repository itself was to be meticulously maintained, with regular and meaningful commits demonstrating the evolution of the project. The final deliverables included a README file in the repository, describing the project's features and providing instructions for running or viewing the website. Overall, the Monthly Website #3 project encompassed research, design, development, and documentation to create a comprehensive and engaging web experience.
